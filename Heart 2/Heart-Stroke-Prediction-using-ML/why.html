Why we made this demo.

Project purpose

This repository is a demonstration for educational purposes. It shows how one might collect intake features, train a Random Forest model locally (see Heart Stroke Prediction.py), and expose a simple frontend for scoring. The current web UI uses a demo scoring heuristic and is not connected to the saved model file by default.

Important disclaimer

The estimator provided here is NOT a medical device. It has not been clinically validated and should not be used to delay or replace professional medical advice. If users are at risk or symptomatic, seek immediate care.

Next steps to productionize

  • Compute and report validation metrics (accuracy, ROC AUC) on holdout data.
  • Expose a secure backend endpoint that loads the trained model for predictions.
  • Perform external validation and clinical review before any deployment.
