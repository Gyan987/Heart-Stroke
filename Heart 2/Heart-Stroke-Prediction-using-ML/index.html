How it works

We collect clinical features, run them through an ML model, and provide an explainable risk estimate.

  1. Enter patient data into the intake form.
  2. Client tries the production backend for a model prediction; falls back to the demo scorer.
  3. Export a PDF report and interpret feature explainability in the dashboard.
