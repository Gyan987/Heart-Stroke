# 📌 Project Overview

**AI Heart Stroke Predictor** is an intelligent healthcare application designed to predict the **risk of heart stroke** using **Machine Learning and AI-based analysis**.

The system collects **health-related data from the user**, processes it through a **trained machine learning model**, and then uses an **LLM (Large Language Model)** to analyze the result and generate a **detailed health risk report**.

This project aims to help users **identify potential stroke risk early** and encourage better lifestyle and healthcare decisions.

---

# 🚀 Features

- Collects health information from users
- Predicts **stroke probability using machine learning**
- Uses **LLM for intelligent health explanation**
- Generates **risk percentage and risk category**
- Provides **AI-generated health recommendations**
- Simple and easy-to-use interface
- Fast prediction results

---

# 🧠 How the System Works

The prediction process follows this pipeline:

### Workflow

1. User enters health details
2. Data is preprocessed
3. Machine learning model predicts stroke risk
4. Risk percentage is calculated
5. LLM generates explanation and health insights
6. Final result is displayed to the user

---

# 📊 User Input Data

The system uses several health parameters such as:

| Parameter | Description |
|-----------|-------------|
| Age | Age of the user |
| Gender | Male / Female |
| Hypertension | High blood pressure |
| Heart Disease | Existing heart disease |
| BMI | Body Mass Index |
| Average Glucose Level | Blood sugar level |
| Smoking Status | Smoking habit |
| Work Type | Type of work |
| Residence Type | Urban / Rural |
| Physical Activity | Exercise level |

# 📈 Output Generated

The system generates:

- Stroke Risk Percentage
- Risk Category (Low / Medium / High)
- AI-generated explanation
- Preventive health recommendations
- Lifestyle improvement suggestions
